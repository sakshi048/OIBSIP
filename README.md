# OIBSIP - Oasis Infobyte Java Internship Projects

Two console-based Java applications developed for Oasis Infobyte internship.

## Projects
1. **Online Reservation System** - Train ticket booking system
2. **Digital Library Management System** - Library book management system

## Technologies
- Java (Core)
- File Handling
- Object-Oriented Programming
- Console-based Interface

## How to Use
1. Clone repository: `git clone https://github.com/sakshi048/OIBSIP.git`
2. Navigate to project folder
3. Compile: `javac *.java`
4. Run: `java Main`
